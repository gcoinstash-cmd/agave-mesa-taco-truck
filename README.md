# Agave & Mesa Storefront

Welcome to the Agave & Mesa taco truck storefront template. This project is a modern, high-performance web application designed for mobile-first taco businesses.

## Features
- Dynamic Menu Management
- Real-time Order Tracking
- Integrated Payment Processing
- Supabase Backend for data persistence

## Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`
3. Configure environment variables in `.env`.
4. Run migrations: `supabase migration up`
5. Start development server: `npm run dev`

## Configuration
This template uses the Agave & Mesa configuration architecture for styling and inventory management.
